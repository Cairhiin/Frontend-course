# Frontend Course - Franciscus van de Voorde
Frontend design course: Coursework and Project

#### RUNNING THE PROJECT
```
git clone https://github.com/Cairhiin/Frontend-course.git

cd portfolio-project

npm install

npm run build

npm install -g serve
 
serve -s build

Open a browser at http://localhost:3000
```
